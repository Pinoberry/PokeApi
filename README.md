# Pokédex Vanilla

Aplicación web sencilla que consume PokeAPI usando únicamente HTML, CSS y JavaScript.

## Características

- Búsqueda por nombre o ID
- Paginación de resultados
- Visualización de tarjetas con imagen oficial y tipos
- Detalle en modal con tipos, habilidades, medidas y movimientos
- Diseño responsive y moderno

## Requisitos

- Navegador moderno
- Conexión a Internet para consumir PokeAPI

## Uso

1. Abrir el archivo `index.html` en el navegador.
2. Usar el buscador para ingresar nombre o ID del Pokémon.
3. Navegar con los botones de paginación.
4. Hacer clic en una tarjeta para ver detalles.

## Estructura

- `index.html`: Estructura de la interfaz y enlace a recursos.
- `styles.css`: Estilos con enfoque responsive.
- `app.js`: Lógica de consumo de API, paginación, búsqueda y modal.

## Notas

- La fuente de datos proviene de [https://pokeapi.co](https://pokeapi.co).
- Si se requiere servir desde un servidor local, puede utilizarse cualquier servidor estático.
